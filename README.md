# $\color{Gold}{PROYECTO\ -\ CRUD-Breeze\ Alvaro\ Gomez}$
</div>

En este repositorio encontraremos mi segunda aplicación creada con Laravel y usando APIs, Breeze y blueprint es la segunda parte de dos sobre un $$\color{RED}{CRUD}$$ creado con Laravel con tematica de personajes de $$\color{RED}{Dragones\ y\ Mazmorras(DnD)}$$.

Se trata de una aplicación simple con una paleta de colores rojo caoba y dorados para seguir con la tematica del famoso juego de mesa. Usa el model de la <a href="https://github.com/OravlaGG/Proyecto-API">parte anterior</a> con la diferencia que uso Breeze para realizar el login y las autentificaciones y uso Blueprint para ahorrar tiempo a la hora de crear los modelos.

<p align="center"><img src="https://github.com/OravlaGG/CRUD-Breeze/blob/main/Capturas/blueprint.png" width="400px"></p>

Como tal es la misma estructura que la anteriro solo que con la sintaxis de Blueprint. Una vez creado el archivo <a href="https://github.com/OravlaGG/CRUD-Breeze/blob/main/draft.yaml">draft.yaml</a> es ejecutarlo y te genera las migraciones, los controladores y los modelos.

<div align="center"> 
<h2>Formato de las páginas</h2>
</div>

El formato de las paginas estan subdivididos por Breeze usando el <x-app-layout> gracias a esto evito la repetición de código para implementar las navbar y los fondos. Tambien me ha habilitado ahorrarme la creación de formularios de creación y edición por separado. Usan el mismo llamado form.blade.php.

<p align="center"><img src="https://github.com/OravlaGG/CRUD-Breeze/blob/main/Capturas/layout.png" width="400px"></p>

Gracias al uso de los frameworks he podido evitar la repetición de código y se consigue rebajar mucho el tiempo de trabajo
